> 本文介绍了类的相关知识

## 类的定义与使用

```c++
//student.h
class CStudent{
public:
  void display();
private:
  int num;
  char name[20];
  int age;
};

//student.cpp
#include <iostream>
#include "student.h"

using namespace std;

void CStudent::display(){
  cout << "num:" << num << endl;
  cout << "name:" << name << endl;
  cout << "age:" << age << endl;
}

//main.cpp
#include <iostream>
#include "student.h"
int main(){
  CStudent stu1;
  stu1.display();
  return 0;
}
```

## 类和struct的区别

1. 类的成员的默认访问权限是private，struct的成员的默认访问权限是public
2. 类的默认继承权限是private，struct的默认继承权限是public
3. 类可以用作模板，而struct不能
